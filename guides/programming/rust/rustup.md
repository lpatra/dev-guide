# RustUp
- update the toolchain
```bash
rustup update
```
- update rustup
```bash
rustup self update
```
- check current toolchain
```bash
rustup show
```
- override toolchain for a particular directory
```bash
rustup override set nightly

# downgrade to an older version
rustup override set nightly-2018-02-21
```
